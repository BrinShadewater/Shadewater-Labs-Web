# Contributing

## Local Setup

```shell
npm ci
npm run dev
```

Before opening a pull request:

```shell
npm run lint
npm run build
```

## Pull Requests

- Keep changes focused.
- Include screenshots for visible UI changes.
- Note any generated assets, synced explainers, or SEO-snapshot changes.
- Preserve the Shadewater Labs voice: practical, sharp, and a little strange around the edges.

## Generated Material

If a change affects generated content, run the relevant sync script and include the generated files in the same change.
