# Shadewater Labs Web

Standalone Vite/React site for [shadewaterlabs.com](https://shadewaterlabs.com/).

## Local Development

```bash
npm ci
npm run dev
```

## Production Build

```bash
npm run build
```

The app serves the Labs landing page at `/` and project pages at:

- `/shadewater-seo-report`
- `/webp-me-daddy`
- `/inkmaster-studio`
