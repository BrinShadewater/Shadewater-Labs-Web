# Project Brief

## Purpose

Shadewater Labs Web is the public portfolio and lab surface for Brin Shadewater. It connects finished projects, active experiments, technical explainers, and proof assets into one coherent identity.

## Audience

- Potential collaborators or clients evaluating quality and range
- People looking for specific tools or project pages
- Search engines and AI answer surfaces parsing the project ecosystem
- Future maintainers returning to the project after time away

## Product Principles

- Show the work quickly.
- Pair atmosphere with concrete proof.
- Explain projects without flattening their voice.
- Keep generated assets traceable.
- Make SEO and metadata visible as craft, not invisible plumbing.

## Current Project Families

- Film and entertainment web surfaces
- SEO and reporting products
- Image conversion and asset-pipeline tooling
- AI-assisted creative production tools
- Personal/brand portfolio material

## High-Risk Areas

- Generated PDFs and snapshots can drift from source scripts.
- Project pages can become stale if deployed projects move.
- Heavy images can slow down a portfolio page if variants are not maintained.
- Copy can become too vague if it loses project-specific proof.
